# Componentes de Texturizado del Proyecto
Ropa: Diseños de indumentaria, uniformes, capas y atuendos representativos de las funciones o profesiones de los habitantes.
Rostros: Mapeos faciales que incluyen expresiones, ojos, bocas y rasgos distintivos para dar personalidad a cada NPC.
Cabello: Variaciones de peinados, tonalidades y estilos capilares adaptados a los diferentes modelos de personajes.
Variaciones por Edad: Texturas específicas para diferenciar visualmente etapas generacionales como jóvenes, adultos y ancianos.
Diseños Personalizados: Estética modular y texturas secundarias para elementos únicos o variantes especiales dentro de la población.
