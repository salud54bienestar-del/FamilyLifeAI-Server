# Minecraft Village Soul
Complemento de Minecraft Bedrock para la integración de Village Soul, encargado de estructurar el comportamiento, los recursos visuales, los NPCs, sus interacciones personalizadas, las animaciones y el puente de comunicación en red con el servidor central de simulación e inteligencia artificial.
Estructura del Complemento
behavior_pack/: Paquete de comportamiento (lógica de entidades, scripts de Script API y eventos).
resource_pack/: Paquete de recursos (texturas, modelos, animaciones y sonidos de los habitantes).
scripts/: Conexión con el servidor externo, gestión del chat y procesamiento de diálogos mediante solicitudes de red.
