🗂️ Estructura del Sistema de Personajes (Village Soul)
Esta documentación resume el núcleo del sistema de habitantes. Con todos los archivos JSON y motores configurados, la estructura general dentro del directorio de tu add-on para Minecraft Bedrock queda organizada de la siguiente manera:

⚙️ Resumen de Módulos Integrados
Identidad y Física: Define la apariencia, herencia genética, etapas de crecimiento y el perfil base de cada habitante.
Sociabilidad y Psicología: Agrupa las personalidades, gustos, preferencias, conversaciones y relaciones para que cada NPC reaccione de forma única.
Evolución y Vida: Controla las habilidades, la progresión por experiencia, los objetivos vitales y la pertenencia a un núcleo familiar intergeneracional.
Entorno: Vincula las acciones cotidianas de los personajes con las ubicaciones físicas de la aldea (escuela, capilla, granja, hogares).
