Sistemas del servidor Village Soul
Aquí se almacenarán los sistemas principales que controlan el funcionamiento del mundo:
Acciones de los habitantes
Eventos del mundo
Emociones
Relaciones
Familias
Profesiones
Memorias

Estos sistemas permitirán que Village Soul tenga un mundo dinámico donde cada habitante pueda actuar, cambiar y evolucionar según sus experiencias.
